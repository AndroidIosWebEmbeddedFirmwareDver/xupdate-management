# xupdate-management

> 使用Vue.js编写的版本更新管理后台，为XUpdate提供版本更新管理

## 运行

``` bash
# install dependencies
npm install

# serve for development with hot reload at http://localhost:8088/mock
npm run dev

# preview for production at http://localhost:1111
npm run preview

```
